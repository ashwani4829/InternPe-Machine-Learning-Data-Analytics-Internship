# Task 4 - Breast Cancer Detection using Data Analytics & Machine Learning

## Objective

Develop a machine learning classification model to detect breast cancer using diagnostic medical data. The project combines data analytics and machine learning techniques to support early disease detection.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Workflow

- Loaded and explored the breast cancer dataset
- Performed data cleaning and preprocessing
- Conducted Exploratory Data Analysis (EDA)
- Visualized feature distributions and correlations
- Selected relevant features for model training
- Built and trained classification models
- Evaluated model performance using classification metrics
- Predicted breast cancer diagnosis

## Results

Successfully developed a classification model capable of detecting breast cancer with high accuracy while gaining valuable insights through data analytics.
