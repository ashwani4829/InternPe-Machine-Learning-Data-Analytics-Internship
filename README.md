# 📊🤖 InternPe Data Analytics & Machine Learning Internship

Welcome to my **InternPe Data Analytics & Machine Learning Internship** repository.

This repository contains the projects I completed during my internship at **InternPe**, where I applied **Data Analytics** and **Machine Learning** techniques to solve real-world business and healthcare problems. The projects focus on data preprocessing, exploratory data analysis (EDA), predictive modeling, model evaluation, and visualization using Python.

---

## 🚀 Internship Projects

### 📌 Task 1 – Diabetes Prediction with Machine Learning
Developed a machine learning model to predict whether a patient is diabetic based on medical diagnostic measurements.

**Key Highlights**
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Selection
- Model Training & Evaluation
- Diabetes Prediction

---

### 📌 Task 2 – Car Price Prediction
Built a regression model to estimate the selling price of used cars based on various vehicle features.

**Key Highlights**
- Data Preprocessing
- Feature Engineering
- Regression Modeling
- Model Evaluation
- Price Prediction

---

### 📌 Task 3 – IPL Winning Team Prediction
Developed a machine learning model to predict the winning team of IPL matches using historical match data.

**Key Highlights**
- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Classification Modeling
- Match Outcome Prediction

---

### 📌 Task 4 – Breast Cancer Detection
Built a classification model to detect breast cancer using diagnostic medical features.

**Key Highlights**
- Data Preprocessing
- Exploratory Data Analysis
- Classification Models
- Performance Evaluation
- Disease Prediction
