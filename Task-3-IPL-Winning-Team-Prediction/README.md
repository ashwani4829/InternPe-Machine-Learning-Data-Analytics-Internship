# Task 3 - IPL Winning Team Prediction using Data Analytics & Machine Learning

## Objective

Develop a machine learning model to predict the winning team of IPL matches using historical match data. The project integrates data analytics with predictive modeling to improve match outcome prediction.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Workflow

- Loaded and explored the IPL dataset
- Performed data cleaning and preprocessing
- Conducted Exploratory Data Analysis (EDA)
- Applied feature engineering
- Visualized match statistics and trends
- Built and trained classification models
- Evaluated model performance
- Predicted winning teams

## Results

Successfully developed a predictive model capable of forecasting IPL match outcomes while uncovering insights from historical match data.
